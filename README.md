This code is for the readers of the book (see https://www.amazon.com/dp/B08PXHJCXY), and for personal use only. For commercial use please reach out (my email is in the book's preface and back cover).

I ask that you please post an honest review of the book on Amazon.com in return. The feedback I got so far for the previous editions has been limited, and I need it to improve the book further. This is hobby project that is worked on weekends/evenings, so any assistance is greatly appreciated.

All reusable code is in .h files, and test*.cpp files allow testing various parts of it.

The code has been reasonably tested, but no guarantees are made about its quality or behavior. I.e. the code is provided "as is", without any implicit warranty of merchantability. So don't use it in pacemakers, nuclear reactors, etc.

Github seems to have a bug in downloading the slides--the best way to get them is to download the entire repository.

## Repository Structure

| Folder | Description |
|--------|-------------|
| [AutoRegressionTest](AutoRegressionTest/) | Autoregressive (AR) time series models and forecasting |
| [Compression](Compression/) | Data compression algorithms: Huffman coding, LZW |
| [ComputationalGeometry](ComputationalGeometry/) | Geometric algorithms: convex hull, K-D trees, polygon area |
| [Cryptography](Cryptography/) | Cryptographic primitives and utilities |
| [ErrorCorrectingCodes](ErrorCorrectingCodes/) | Error detection/correction: CRC, LDPC, Reed-Solomon |
| [ExternalMemoryAlgorithms](ExternalMemoryAlgorithms/) | Algorithms for data sets larger than RAM: external B-trees, CSV/file I/O |
| [FinancialCalculations](FinancialCalculations/) | Financial math: annuities, cash flows, option pricing, portfolio optimization |
| [Graphs](Graphs/) | Graph algorithms: traversal, shortest paths, network flow |
| [HashTable](HashTable/) | Hash table implementations: chaining, linear probing, Bloom filters |
| [Heaps](Heaps/) | Heap data structures: binary heap, indexed heap |
| [LargeNumbers](LargeNumbers/) | Arbitrary-precision integer and rational arithmetic |
| [MachineLearning](MachineLearning/) | Machine learning: classification, regression, clustering, neural networks, SVM, and more |
| [MiscAlgs](MiscAlgs/) | Miscellaneous algorithms: Fenwick tree, prime sieve, LRU cache, combinatorics |
| [NumericalMethods](NumericalMethods/) | Numerical analysis: matrices, FFT, integration, interpolation, ODE solvers |
| [NumericalOptimization](NumericalOptimization/) | Continuous optimization: L-BFGS, Nelder-Mead, differential evolution, linear programming |
| [Optimization](Optimization/) | Combinatorial optimization: knapsack, bin packing, SAT, meta-heuristics |
| [RandomNumberGeneration](RandomNumberGeneration/) | RNG, statistics, MCMC, bootstrapping, time series, hypothesis testing |
| [RandomTreap](RandomTreap/) | Randomized search structures: treaps, skip lists, tries |
| [Scrap](Scrap/) | Experimental and auxiliary algorithms not yet fully integrated |
| [Slides](Slides/) | Presentation slides: intro to machine learning, Monte Carlo algorithms |
| [Sorting](Sorting/) | Sorting algorithms (comparison and non-comparison based) |
| [StringAlgorithms](StringAlgorithms/) | String algorithms: substring search, suffix arrays, regex, diff, rank/select |
| [Utils](Utils/) | Core utilities: vector, stack, queue, union-find, bitset, debugging helpers |
